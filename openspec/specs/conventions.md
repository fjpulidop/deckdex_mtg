# Conventions

Python 3.9+, type hints, PEP-8. Lint/format: ruff (optional black); type-check: mypy. Tests: pytest/unittest; mock Scryfall, Sheets, OpenAI. Logging: stdlib logging; --verbose/--quiet. Commits: short imperative summary; branches feat/ fix/ chore/ docs/. Doc sheet column changes in [data-model](data-model.md). No secrets in repo; rotate keys; least-privilege Service Account.
