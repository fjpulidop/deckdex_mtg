# Persona: MTG Collector

> Dedicated collector who tracks value, condition, and set completion across years of accumulation.

## Profile

| Field | Value |
|-------|-------|
| **Name** | "Morgan" — The Portfolio Collector |
| **Age** | 28-50 |
| **Collection size** | 5,000-50,000+ cards |
| **Monthly spend** | $100-500+ on sealed product and singles |
| **Focus** | Set completion, premium variants, long-term value |
| **Tools used** | TCGPlayer, EchoMTG, MTGStocks, MTGGoldfish, spreadsheets, Dragon Shield scanner |
| **Mindset** | Treats collection as both hobby and investment portfolio |

## Behaviors

- **Acquisition cycle**: Opens sealed product (Collector Boosters preferred) → sorts by set/condition → catalogs into tracking system → identifies set completion gaps → buys targeted singles.
- **Price monitoring**: Checks daily movers/shakers on MTGStocks/MTGGoldfish. Sets price alerts for want-list cards. Monitors total collection value weekly.
- **Condition grading**: Evaluates every card's condition (NM/LP/MP/HP/DMG). High-value cards considered for CGC/PSA grading. Stores in penny sleeves + toploaders + binders by set.
- **Trading**: Maintains trade binders (physical + digital lists). Cross-references values before trades.
- **Set completion**: Tracks % complete per set, separately for regular, foil, and special treatments.

### Pains

| Severity | Pain |
|----------|------|
| **Critical** | Variant explosion — a single card has 5-7+ versions (regular, foil, extended art, borderless, showcase, foil-etched, serialized). No tool cleanly aggregates "I have 3 of 7 variants" |
| **Critical** | No unified collection value dashboard — must cross-reference EchoMTG + TCGPlayer + MTGStocks + spreadsheets |
| **High** | Card scanning inaccuracy — Dragon Shield can't reliably distinguish art variants, borderless vs regular, alternate frames |
| **High** | Condition tracking gaps — most apps track quantity and foil/non-foil but don't handle per-copy condition grades. A NM and LP copy have very different values |
| **High** | Price alert fragmentation — TCGPriceAlert limits free plan to 15 cards, MTGStocks requires premium subscription |
| **Medium** | Product fatigue — 7+ major releases/year, each with exclusive Collector Booster treatments |
| **Medium** | Serialized card tracking — no standardized way to note serial numbers |
| **Low** | EV calculation difficulty for sealed product with daily market fluctuations |

### Gains

| Impact | Gain |
|--------|------|
| **High** | One-stop portfolio dashboard: total value, daily/weekly/monthly change, breakdown by category |
| **High** | Accurate scanning that distinguishes all treatments and printings |
| **High** | Automated set completion tracking with visual progress per variant type |
| **Medium** | Price alerts across multiple marketplaces (TCGPlayer + Cardmarket + eBay) in one system |
| **Medium** | Export for insurance documentation with condition and current market value per card |
| **Medium** | Historical price charts with "I bought at X" markers |
| **Low** | EV calculator for sealed product based on current market data |

## Key Insight

> The collector's #1 frustration is **variant proliferation breaking every tracking tool**. When a single card exists in 7 printings across 3 sets, condition matters per copy, and prices differ 10x between variants — spreadsheets become the only "tool" that works. DeckDex can differentiate by treating variants and conditions as first-class entities, not afterthoughts.
