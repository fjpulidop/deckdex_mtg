# Persona: MTG Player

> Commander/EDH enthusiast who plays weekly and manages multiple decks.

## Profile

| Field | Value |
|-------|-------|
| **Name** | "Alex" — The Commander Regular |
| **Age** | 25-40 |
| **Play frequency** | 2-4 sessions/month at LGS or friends' homes |
| **Decks** | 5-15 Commander decks simultaneously |
| **Monthly spend** | $50-200 on singles and sealed product |
| **Formats** | Commander (primary), occasionally Draft/Sealed, dabbles in Standard/Modern |
| **Tools used** | Moxfield, EDHREC, Scryfall, manabase.gg, TCGPlayer, YouTube, Reddit (r/EDH) |

## Behaviors

- **Deck building cycle**: Discovers commander (spoilers, EDHREC, content creators) → theorycrafts on Moxfield → checks prices on TCGPlayer → orders singles → iterates after playtesting.
- **Game night prep**: Picks deck based on group power level, physically swaps expensive staples between decks, checks bracket rating.
- **New set evaluation**: Scans spoilers for Commander staples, checks EDHREC "New Cards" page, evaluates upgrades across multiple existing decks.
- **Uses 4-6 tools simultaneously**: No single platform does everything they need.

## Value Proposition Canvas

### Customer Jobs

| Type | Job |
|------|-----|
| **Functional** | Build and iterate Commander decks with accurate data (prices, legality, mana math) |
| **Functional** | Know which cards I own, which decks they're in, and which I need to buy |
| **Functional** | Prepare for game night quickly — right deck for the power level, cards physically ready |
| **Functional** | Discover relevant new cards from new sets for my existing decks |
| **Functional** | Optimize mana bases without exporting between separate tools |
| **Social** | Communicate deck power level clearly to my playgroup (bracket system) |
| **Social** | Share deck lists and get feedback from the community |
| **Emotional** | Feel confident my deck is well-built before sitting down to play |

### Pains

| Severity | Pain |
|----------|------|
| **Critical** | Spending 30+ minutes swapping expensive staples (fetch lands, Mana Crypt, The Great Henge) between decks before game night |
| **Critical** | No cross-deck view of "owned vs. needed" — buying cards I already own because I forgot which deck they're in |
| **High** | Building a deck online only to realize half the cards are committed to other decks |
| **High** | Tool fragmentation — juggling Scryfall + EDHREC + Moxfield + manabase.gg + TCGPlayer for one workflow |
| **High** | New set overwhelm — 7+ products/year, evaluating upgrades across 10+ decks manually |
| **Medium** | Power level / bracket confusion — no automated bracket estimation |
| **Medium** | Mana base building is guesswork without integrated optimization |
| **Low** | Playtester UX issues in existing tools (Archidekt battlefield clutter, Moxfield single-player only) |

### Gains

| Impact | Gain |
|--------|------|
| **High** | Single platform combining deck building + collection tracking + pricing + mana optimization |
| **High** | Visual cross-deck card allocation — see at a glance where each copy is assigned |
| **High** | Automated "upgrade suggestions" when new sets release, personalized to existing decks |
| **Medium** | Quick bracket/power-level estimation for any deck list |
| **Medium** | Mobile-friendly experience for use at the LGS or game table |
| **Low** | Integrated playtesting without exporting to another tool |

## Key Insight

> The #1 unmet need across the MTG player community is **"where is this card right now?"** — the intersection of collection tracking and deck management that no tool handles well. Moxfield is great for deck building but weak on collection. EchoMTG is great for collection but weak on decks. DeckDex can own this intersection.
